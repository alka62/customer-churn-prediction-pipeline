# Feature Engineering Documentation

Created the following new features:

1. feature_sum
   - Sum of first two numerical columns

2. feature_diff
   - Difference between first two numerical columns

3. feature_product
   - Product of first two numerical columns

4. feature_ratio
   - Ratio of first two numerical columns

5. feature_square
   - Square of first numerical column

These engineered features help improve model performance by extracting more information from existing data.
